## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/perspective-ui.git
cd perspective-ui
```

2. Install dependencies:
```bash
cd src
npm install
```

### Development

To start the development server:
```bash
cd src
npm run start
```

To build the application:
```bash
cd src
npm run build
```
