git clone https://github.com/jefking/perspective-ui.git
cd perspective-ui
```

2. Install dependencies:
```bash
npm install
```

3. Development:
- For Electron development:
```bash
npm run dev:electron
```
- For web-only development (some features like file reading won't work):
```bash
npm run dev:web
```

4. Build:
```bash
npm run build
